## Run Locally

Clone the project

```bash
  git clone https://github.com/ruturajmengal2016/assesment-next.git
```

Go to the project directory

```bash
  cd assesment-next
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
